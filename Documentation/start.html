Quick Start

Requirements

Server :
  • PHP 7.3
  • If you plan to upload large files please increase you server upload_max_filesize

1 - Go to Select PHP version

2 -Update information

Upload files and create database

1 - Login to your cpanel and click MYSQL DATABASE

2 - Enter database name

3 - Create new database user

4 - Choose database name and database user then click add

5 - choose All privileges then click make changes

6 - Now go to your hosting file manager

7 - Choose your directory

8 - Click upload

9 - Click select then select the Swipgle.zip

10 - Choose Swipgle.zip then click extract

11 - Click extract files

Now your database and files is ready open your website URL and follow installation steps.

Installation

Step 1 ) - Enter Your database information.

Step 2 ) - Click Get Started.

Step 3 ) - Enter your webiste information Please Make Sure That URL is Correct.

Step 4 ) - Enter admin information.

    Setup cron job

    Now you need to setUp cron job to delete expiry files

    1 - Go to Cron Jobs

    2 - SetUp it like this

    /usr/local/bin/php /home/SERVER_PATH/Application/artisan schedule:run >> /dev/null 2>&1
  • Change SERVER_PATH to your website files directory

    • Setup SMTP

    1 - Go to your admin panel settings>SMTP and enter your SMTP information

    Setup Paypal payments

    1 - Go to paypal developer section https://developer.paypal.com/home and log in

  • Go to My apps
  • Choose live
  • Click create app

    • 2 - Enter app name and click create app

    3 - This is your app information

    4 - Go to admin and settings then payments gateway and enter your information

    Setup Amazon s3
